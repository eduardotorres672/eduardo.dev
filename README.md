# Eduardo Cosi — Portfólio

Portfólio pessoal desenvolvido com HTML, CSS e JavaScript puro — sem frameworks, sem dependências.

🔗 **https://eduardo-dev-six.vercel.app/**

---

## Sobre

Site portfólio com seções de projetos, conhecimentos técnicos, experiência profissional e contato.  
Design minimalista inspirado em portfólios de desenvolvedores modernos.

## Stack

- HTML5 semântico
- CSS3 — Grid, Flexbox, variáveis CSS, animações
- JavaScript vanilla — Intersection Observer, menu acessível, relógio local
